# PDF Examination

Examing a PDF file to find information and hidden information from redacted

# Tools

- Metadata2go
- Adobe Acrobat Free PDF editor

## Using Metadata2go

When importing a file on metadata2go, it will give a variety of specific information

For this example, inserting api.pdf file reveal several information that include:
- creator tool
- pdf version
- file size
- date created

There are more information but that is just a few example shown.

## Removing Redacted in PDF

Google search Adobe Acrobat Free PDF editor and click the first link

Insert pdf file, for example api.pdf

On the top left corner click Edit

Right click on the PDF and select all and delete

Click the undo button on top or CTRL-Z

PDF is no longer redacted

Another method without PDF Editor is copy all text in PDF file and paste it in a document
